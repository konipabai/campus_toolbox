# 校园百宝箱

校园百宝箱是一个为学校提供便利和实用工具的开箱即用校园应用项目。



## 功能

**①** 预约教室

**②** 预约图书馆座位

**③** 预约球场

**④** 故障报修

**⑤** 失物招领

**⑥** 校园招聘

**⑦** 校园新闻



## 技术栈

前端：Vue3，Typescript，Vue Router，Pinia，Vite，Axios，Element-plus，Echarts

后端：Nestjs，Typescript，Typeorm

数据库：Mysql
