# 校园百宝箱（持续更新中）

校园百宝箱是一个为学校提供便利和实用工具的开箱即用校园应用项目。



## 功能

**①** 预约教室（完成）

**②** 预约图书馆座位（完成）

**③** 预约球场

**④** 故障报修

**⑤** 失物招领（完成）

**⑥** 校园招聘

**⑦** 校园新闻



## 使用技术（后续完善项目会继续添加新技术）

前端：Vue3，Typescript，Vue Router，Pinia，Vite，Less，Axios，Element-plus，Echarts

后端：Nestjs，Typescript，Typeorm

数据库：Mysql
